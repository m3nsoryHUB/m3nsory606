# Музыкальный плеер с эквалайзером

Веб-плеер для прослушивания музыки с 10-полосным графическим эквалайзером на Web Audio API.

## Возможности

- Воспроизведение локальных аудиофайлов (MP3, WAV, OGG, FLAC и др.)
- 10-полосный эквалайзер (от 60 Гц до 16 кГц) на BiquadFilter
- Готовые пресеты: Рок, Поп, Джаз, Классика, Электроника, Бас-буст, Вокал
- Плейлист с возможностью добавления и удаления треков
- Спектральный визуализатор в реальном времени
- Управление: воспроизведение / пауза / следующий / предыдущий / громкость / перемотка
- Горячие клавиши: `Space` — пауза, `Shift+→/←` — следующий/предыдущий

## Запуск

Откройте `index.html` в браузере, либо запустите простой статический сервер:

```bash
python3 -m http.server 8000
# затем откройте http://localhost:8000
```

## Файлы

- `index.html` — разметка
- `styles.css` — стили
- `player.js` — логика плеера, эквалайзер и визуализатор
